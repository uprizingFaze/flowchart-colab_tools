#### Comenzar:

#####  Clonar el repositorio:

```bash
git clone https://github.com/uprizingFaze/flowchart-colab_tools.git
cd flowchart-colab_tools
```

#####  Instalación de dependencias:


```bash
npm install
```

##### Ejectute el servidor de desarrollo:


```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```
